Player cards have wrapper .dealer-player-cards. If there is more than 3 cards they will get class name .compact.

A player card contains a header .dealer-player-card-header and multiple .dealer-player-card-data-row elements with data label and value pairs.

.dealer-player-card-data-row elements can have additional class names to indicate different states: .empty, .highlight-highest, .highlight-win, .highlight-alert, .insurance-row and .highlight-insurance.

Possible backbetters on bets are indicated with .dealer-player-bet-data-indicator elements.
