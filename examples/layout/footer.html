Fixed footer

Footer is fixed to the bottom of the screen and has high enough z-index to keep it on top of most elements.

When using the footer, body should have class name .has-footer. This gives .main-content the same amount of bottom padding as the height of the footer so that content wouldn't be hidden behind the footer.
