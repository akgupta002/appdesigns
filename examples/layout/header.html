Header is fixed to the top of the screen and has high enough z-index to keep it on top of most elements.

When using the header, body should have class name .has-header. This gives .main-content the same amount of top padding as the height of the header so that content wouldn't be hidden behind the header.

    Casino Alerts component is basically same as sub-header component Table Alerts. It consists of a dropdown with custom content.

    .dropdown-alert-filter-link elements control the alerts content visibility. They need to have data-target attributes matching the IDs of the content elements.

    The content elements .dropdown-alert-filters also have class names .collapse, which initially hides the content. The content that is supposed to be visible has class name .in.

    The alert contents .dropdown-alert-box-expandable are expandable themselves also. .dropdown-alert-box-expand-link elements control the expansion of them in the same manner with class name .collapse and .in.

    Requires alerts.js.
