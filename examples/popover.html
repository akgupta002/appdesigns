Popover

Wrapper can be any element, but it needs to have class name .dropdown. Toggle element needs to contain attribure data-toggle="dropdown" and be located inside the wrapper.

Possible positioning classes for the wrapper: .dropdown-center, .dropdown-right, .dropside, .dropside-right and .dropup. These classes position the dropdown element relative to the wrapper.

Dropdown closes from a click outside the dropdown or from an element inside the wrapper with class name .btn-close-dropdown

Requires dropdowns.js
